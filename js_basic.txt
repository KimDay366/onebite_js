1. 내장함수와 메서드
- 내장함수 : 자바스크리븥에 기본으로 포함 된 독립함수로, 단독 호출이 가능함.
            [ex] alert();, setTimeout();
- 메서드 : 어떤 객체에 속한 함수로, 특정 객체 뒤에 붙는 함수. 따라서 단독 호출이 불가능함.
            [ex] arr.push(); Array.isArray(); 

            - 정적 메서드 : 이미 자바스크립트 자체에 선언 된 존재를 적어야 함.
                           [ex] Array.isArray(); 

            - 인스턴스 메서드 : 사용자가 만든 존재에 사용 가능한 메서드. 
                           [ex] qwerArr.push();
